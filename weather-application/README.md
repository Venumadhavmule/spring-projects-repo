# 🌤️ Weather Application

This is a **Spring Boot-based Weather Application** that fetches real-time weather data from the **OpenWeatherMap API** and presents it on a clean, interactive user interface.

---

## 🚀 Features

- **Real-time Weather Data**: Retrieve current weather information for any city.
- **Information Displayed**:
  - 🌡️ Temperature
  - ☁️ Weather conditions (e.g., sunny, cloudy, rainy)
  - 💧 Humidity
- **User-Friendly UI**: A simple and intuitive interface for a smooth user experience.

---

## 🛠️ Technologies Used

- **Backend**: Spring Boot (REST API)
- **Frontend**: HTML, CSS, JavaScript
- **API Integration**: OpenWeatherMap API

---

## ✅ Prerequisites

Before running the project, ensure you have:

1. **Java (JDK 8+)**: [Download Java](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html)
2. **Maven**: [Install Maven](https://maven.apache.org/)
3. **API Key** from OpenWeatherMap: [Get API Key](https://openweathermap.org/api)
